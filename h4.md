## a) Install daemon of your choice manually

I chose to download HAProxy which was mentioned in the [homework assignment](https://terokarvinen.com/palvelinten-hallinta/#laksyt). 
Haproxy is a reverse proxy and reverse proxy is a server that operates between your server/servers and a visitor of your server adding security and other usefull features like load balancing. You can read more about reverse proxies from the [geeksforgeeks article](https://www.geeksforgeeks.org/computer-networks/what-is-a-reverse-proxy/).

So first step of course was to install the daemon `sudo apt-get install haproxy`. Then I had to make sure that it was installed correctly so I ran few commands to test it out. First I ran the command `apt list --installed | grep haproxy` using the package manager and searching for a specific packahe with `grep`.

<img width="789" height="143" alt="image" src="https://github.com/user-attachments/assets/b3a70997-975e-4471-9463-4cc8e9e35c1d" />

I also wanted to try new commands to see if the installation was successfull. So I googled a bit and found the command `sudo haproxy -c -V -f /etc/haproxy/haproxy.cfg` from [haproxy.com](https://www.haproxy.com/blog/testing-your-haproxy-configuration).
The command validates the config and gives output which tells you if there are any problems.
