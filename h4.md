## a) Install daemon of your choice manually

I chose to download HAProxy which was mentioned in the [homework assignment](https://terokarvinen.com/palvelinten-hallinta/#laksyt). 
Haproxy is a reverse proxy and reverse proxy is a server that operates between your server/servers and a visitor of your server adding security and other usefull features like load balancing. You can read more about reverse proxies from the [geeksforgeeks article](https://www.geeksforgeeks.org/computer-networks/what-is-a-reverse-proxy/).
