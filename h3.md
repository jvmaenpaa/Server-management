## x) Read & summarize

In the articles I have listed below, handlers and ansible.builtin.service module are introduced. Handlers are tasks that run only when something has changed, like a config file of a service. And the ansible.builtin.service is used by the handlers to manage a service. 
Let´s imagine that I have just done some changes to a apache2 configuration file and I would like to restart it by using a handler. The handler would use the ansible.builtin.service module to make the restart happen. Of course there is a lot more to it than what I just said but I hope this gives you the basic idea of what these components can do.
